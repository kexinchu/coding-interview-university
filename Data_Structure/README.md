## 必考知识

- 时间复杂度：摊还、主定理、递归式
- 常见数据结构：
    - Heap / Priority Queue（面试重点）
    - HashMap（冲突、rehash、复杂度）
    - Tree / BST / AVL / Red-Black 属性
    - 并查集（union find）
    - 滑动窗口（LeetCode 高频）
    - 单调栈 / 单调队列
- 图算法：
    - BFS / DFS
    - Dijkstra / A*
    - Topological Sorting
    - Strongly Connected Components

## 🎯 高频例题（必刷）
- 滑动窗口最大值
- K 个有序链表合并
- LRU cache（手写）
- 最大子数组和
- 前 K 大元素（heap + quickselect）
- 拓扑排序判断有向图是否有环
- Trie 设计（字符串匹配）
