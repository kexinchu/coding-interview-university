### init函数
- 程序中的所有init方法都会在main函数启动前被调用


### 导入包
- 导入未引用的包时，golang会报错
- 可以使用 "_" 来避免这种报错 <当然不导入未使用的包最好>
```
import (
    "fmt"
    _"strings"
)
```
- 导入的包
  - 标准库中的包会在安装GO的位置找到
  - Go开发者创建的包会在 GOPATH 环境变量指定的目录中查找
- 命名导入
  - 导入相同 名称 的包
```
import (
    "fmt"
    myfmt "mylib/fmt"
)

func main(){
    fmt.Println("Standard Library")

    myfmt.Println("mylib/fmt")
}
```
