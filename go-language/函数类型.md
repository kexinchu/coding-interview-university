### 函数类型

函数是一种类型：
```
type Add func(a int, b int) int
```
它可以用在任何地方 -- 作为字段类型，参数或者返回值。

```
package main

import (
  "fmt"
)

type Add func(a int, b int) int

func main() {
  fmt.Println(process(func(a int, b int) int{
      return a + b
  }))
}

func process(adder Add) int {
  return adder(1, 2)
}
```

这样使用函数会帮助我们从具体实现中解耦代码，更像在使用接口实现。
